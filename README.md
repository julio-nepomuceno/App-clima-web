# 🌤️ App de Clima

App de clima simples e direto, desenvolvido com **HTML, CSS e JavaScript puro**, integrado à API da OpenWeather para exibir a temperatura atual de qualquer cidade em tempo real.

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript
- [OpenWeather API](https://openweathermap.org/api)
