# 🌤️ App de Clima

App de clima simples e direto, desenvolvido com **HTML, CSS e JavaScript puro**, integrado à API da OpenWeather para exibir a temperatura atual de qualquer cidade em tempo real.

## 🚀 Tecnologias

- HTML
- CSS
- JavaScript
- [OpenWeather API](https://openweathermap.org/api)

## 🔗 Demo

👉 [Clique aqui para testar](https://julio-nepomuceno.github.io/App-clima-web)

## ⚙️ Como rodar localmente

1. Clone o repositório
2. Crie um arquivo `config.js` na raiz com sua chave da OpenWeather:
```js
const API_KEY = "sua_chave_aqui";
```
3. Abra o `index.html` no navegador
